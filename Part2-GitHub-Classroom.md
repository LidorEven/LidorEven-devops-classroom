# Part 2 — Accept the GitHub Classroom Assignment

Your instructor will share a **GitHub Classroom invitation link** in class.

---

## Steps

1. Click the invitation link.
2. Sign in with the GitHub account you created in [Part 1](Part1-GitHub-Account.md).
3. Accept the assignment. GitHub Classroom will automatically create a **private repository** for your pair under the course organisation.
4. Both partners must join the same team/group when prompted, so you share one repository.

After accepting, note your repository URL — it will look like:

```
https://github.com/<org>/<assignment-name>-<your-team>
```
