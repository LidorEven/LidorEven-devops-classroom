# Part 3 — Clone the Repository

Clone the repository to your local machine using SSH:

```bash
git clone git@github.com:<org>/<your-repo-name>.git
cd <your-repo-name>
```

Confirm everything is set up:

```bash
git status
git log --oneline
```
